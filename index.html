About Me

I recently completed my Ph.D. in Computer Science at Peking University, advised by Prof. Yingfei Xiong. My research focused on program generation, program understanding, pre-trained large models, and natural language processing.

I developed novel techniques for neural program repair, code generation, and code retrieval, published in top-tier conferences such as ICSE, IJCAI, and AAAI. My work on type-aware neural program repair and syntax-guided code generation has significantly advanced the state-of-the-art in automated software engineering.

Currently, I am a researcher at DeepSeek, where I focus on developing advanced large models for code generation and logical reasoning. My work aims to bridge the gap between human intent and machine execution, enabling more efficient and intelligent software development.

I am passionate about creating models that assist developers in writing better code, reducing errors, and improving productivity.
